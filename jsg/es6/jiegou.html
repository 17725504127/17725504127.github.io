    # 解构赋值

    # 一、解构赋值的定义：

    es6允许按照一定模式，从数组或者对象中取值，然后把取出来的值赋值给变量，称作解构赋值

    # 二、解构赋值的方式：

    1.数组的解构赋值：
    数组的解构赋值是有次序的，按照数组的的顺序一一解构，如果数组的值少于赋值的变量，那么打印这个变量是undefined,这种情况可以给变量一个默认值

    const arr=["aaa","bbb","ccc]
var[itm1,itm2,itm3="ccc",itm4="ddd"]=arr
console.log(itm1)//aaa
console.log(itm2)//bbb
console.log(itm3)//ccc
console.log(itm4)//ddd

    2.对象的解构赋值：
    对象的解构赋值是没有顺序的，它是根据对象中的key进行解构的，只要变量名和key相同就能解构赋值

     let obj={
        clicked:true,
        loading:false,
        disable:true,
        name:"jixue",
        num:20
        }
 var {clicked,loading,disable,name,num,age="22"}=obj
        console.log(clicked)
        console.log(num)
        console.log(age)

    3.字符串的解构赋值： 字符串的解构赋值跟数组一样，按照顺序一一解构

    var str="123"
var [a,b,c,d]=str
console.log(a)//1
console.log(b)//2
console.log(c)//3
console.log(d)//undefined

    4.函数的解构赋值：
    函数的解构赋值是函数在调用的时候，表面上传的是一个数组或者对象，但是在传进来的那一刻会通过解构赋值的模式对应展开成函数的参数，在函数内部，函数感受到的是展开后的参数,但如果函数调用的时候不传参，那么就会报错，所以可以给整个参数一个默认值

     function show([h1,h2=2,h3]){
        console.log(h1)//1
        console.log(h2)//2
        console.log(h3)//undefined
    }
    show([1])
 function add({name=5,num2=90}={}){
        console.log(name)//5
        console.log(num2)//90
    }
    add()

    # 三、解构赋值的应用：

    1.解析JSON数据：

    var json={head:1,body:2,footer:3}
var {head,body,footer}=json
console.log(head)//1
console.log(body)//2
console.log(footer)//3

    2.交换俩个变量：

    var a = 1
var b = 2
var [b,a] = [1,2]
console.log(a)//2
console.log(b)//1

    3.函数返回的一组数据可以变成单独的：

    function aaa() {
    return [1, 2, 3];
}
[a, b, c] = aaa()
console.log(a)//1
console.log(b)//2
