    # es6新增数组方法

    # 一、map方法：

    map方法返回一个新的数组，数组中的元素为原始数组调用函数处理后的值，map方法不会检测空数组，并且不会改变原数组
    用法：数组.map(function(item,index,arr){return})
    ---参数1: 数组每一项
    ---参数2：索引
    ---参数3：原数组

    var arr=[1,2,3,4,5]
let arr1=arr.map(function(item,index,arr){
    return --item
})
console.log(arr)//[1,2,3,4,5]
console.log(arr1)//[0,1,2,3,4]

    # 二、forEach方法：

    forEach方法用于调用数组的每个元素，将元素传给回调函数,forEach对于空数组是不会调用回调函数的,并且没有返回值 用法：数组.forEach(function(item,index,arr){})
    ---参数1: 数组每一项
    ---参数2：索引
    ---参数3：原数组

    var arr=[1,2,3,4,5]
var sum =0;
var str = arr.forEach(function(item){
   sum+= item;
console.log("sum的值为：",sum); //sum的值为：1 sum的值为： 3 ...
})
console.log(str)//undefined

    # 三、filter方法：

    过滤数组的元素，只要是符合条件的元素就返回到新数组里 用法：数组.filter(function(item,index,arr){return})
    ---参数1: 数组每一项
    ---参数2：索引
    ---参数3：原数组

    var arr=[1,2,90,87,5,20,60,5,9]
var arr1=arr.filter(function(item){
    return item>10
})
console.log(arr1)//[90,87,20,60]

    # 四、every方法：

    检测数组所有的元素是否符合条件,只要有一个元素不符合就返回false 用法：数组.every(function(item,index,arr){return})
    ---参数1: 数组每一项
    ---参数2：索引
    ---参数3：原数组

     var arr=[1,9,10,5,8,20,21]
var arr1=arr.every(function(item){
   return item>10
})
var arr2=arr.every(function(item){
   return item<30
})
console.log(arr1)//false
console.log(arr2)//true

    # 五、some方法：

    检测数组的元素，只有有一个元素符合条件，就返回true 用法：数组.some(function(item,index,arr){return})
    ---参数1: 数组每一项
    ---参数2：索引
    ---参数3：原数组

    var arr=[1,2,3,4,5]
var arr1=arr.some(function(item){
   return item>3
})
console.log(arr1)//true

    # 六、find方法：

    方法返回数组中符合条件的第一个元素。没有符合条件的就返回undefined 用法：数组.find(function(item,index,arr){return})
    ---参数1: 数组每一项
    ---参数2：索引
    ---参数3：原数组

    var arr=[1,2,3,4,5]
var arr1=arr.find(function(item){
   return item>3
})
console.log(arr1)//4

    # 七、findIndex方法：

    方法返回数组中符合条件的第一个元素的索引。没有符合条件的就返回-1 用法：数组.findIndex(function(item,index,arr){return})
    ---参数1: 数组每一项
    ---参数2：索引
    ---参数3：原数组

    var arr=[1,2,3,4,5]
var arr1=arr.findIndex(function(item){
   return item>3
})
console.log(arr1)//3

    # 八、from方法：

    用来将伪数组对象和可遍历的对象转为真数组 用法：Array.from(obj)

    //伪数组对象必须有length属性，如果没有就会转化成空数组
var obj={
      "0":1,
      "1":2,
      "2":3,
      length:3
}
var arr=Array.from(obj)
console.log(arr)//[1,2,3]

    # 九、of方法：

    用来将of方法函数获取到的实参拼成一个数组 用法：Array.of(element)

    let ofArr= Array.of(1,2,3,"aaa",true)
console.log(ofArr)//[1,2,3,"aaa",true]

    # 十、reduce方法：

    这个方法接受一个函数作为参数，这个函数也称做函数累加器,把数组里的每一项从左往右依次递减，直到最后依次运算，最终返回一个结果,可用于数组求和
    用法：数组.reduce(function(prev,next,index,arr){return},初始值)初始值---是prev的初始值
    ---参数1-prev：参与运算的初始值，或者上一次运算返回的结果（第一次会是数组第一个值，第二次取的就是数组第一个值和第二个值的和，依次类推）
    ---参数2-next：依次参与运算的数组每一项（第一次取的是数组第二个值，因为数组第一个值被prev占用了）
    ---参数3-index: 索引----算的是next的索引
    ---参数4-arr : 原数组

    var arr=[1,2,3,4]
var sum=arr.reduce(function(prev,next,item,arr){
      return prev+next
})
console.log(sum)//10
